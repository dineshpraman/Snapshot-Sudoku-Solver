SNAPSHOT SUDOKU SOLVER 

Compatible with Open CV 2.0 above

Coded and Tested in Ubuntu 10.10 Linux

Works with Image sizes in the order of 350 X 350 to 700 X 700 

Compile the code by running  $bash compile.sh in your terminal. Make sure that the path of the installed libraries in compile.sh file are correct.  

Execute the code by $./code sudoku.jpg
(where sudoku.jpg the name of the input sudoku image)
